# dnd-sheet
A local-first D&D 5e character sheet web app (NiceGUI), with spells, companions, and player-safe bestiary.

## Requisiti
- Python 3.11+ (consigliato: la versione del progetto)
- SQLite (incluso in Python)

## Attribuzione SRD
Questo progetto include materiale tratto dallo SRD 5.1. Vedi: [db/datasets/ATTRIBUTION_SRD.md](db/datasets/ATTRIBUTION_SRD.md)


## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
